# Context-Aware Chatbot Using RAG and Google Gemini

A professional-grade, **context-aware chatbot** leveraging **Retrieval-Augmented Generation (RAG)**. The chatbot can answer questions using a **custom document corpus**, while maintaining conversational context.

It uses:
- **FAISS** for vector similarity search
- **HuggingFace embeddings** (`all-MiniLM-L6-v2`)  
- **Google Gemini** for natural language generation
- **Streamlit** for a clean and interactive UI

---

## Features

- Context-aware conversation with memory
- Retrieval from custom text datasets
- Fast similarity search using FAISS
- Google Gemini LLM integration
- Interactive Streamlit frontend for real-time chatting
- Easy deployment and extension
